1. Générer l'application
npx create-react-app kasa
cd kasa
npm start


2. Installer le compilateur Sass
npm install -g sass

3. Structure de l'app
├───assets
│   └─── Images et fichiers
├───components
│   └─── Composants fonctionnels
├───datas
│   └─── Fichier de données au format JSON pour simuler une API
└───styles
   └─── Fichiers de style par composant
